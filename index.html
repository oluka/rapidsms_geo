rapidsms_geo by mossplix

a geographical mapping site for rapidsms

this provides 2 bits of functionality 1.a maps package for rendering maps with mapnik 2.a utils module for providing various utilitie for dealing with the geospatial data 3.an addresses module for automatically detecting addresses 4.a point picker widget 5.api for interfacing with the data Background ========== it also requires http://code.google.com/p/mapnik-utils which includes cascednik for styling of the maps and ogcserver as well as tilecahe you will be responsible for populating your own geo spatial database which in this case is postgis but we plan to add support for spatialite right now we assume you are using postgis
